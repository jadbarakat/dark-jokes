# dark-jokes

Mobile app made with React Native to surface [Sv443's JokeAPI](https://sv443.net/jokeapi/v2/)'s collection of dark jokes.

Features:
- Get a random dark joke
- Filter jokes by category
- Share joke using device native share functionality
- Add joke to favourites
- Sort favourites by date added ascending or descending
- Supports light and dark modes
- Change app tint color with one tap
- "Minimal mode"

Interesting dependencies:
- [expo](https://docs.expo.dev/)
- [react-navigation](https://reactnavigation.org/)
- [jotai](https://jotai.org/)
- [gorhom/bottom-sheet](https://gorhom.github.io/react-native-bottom-sheet/)
- [react-native-toast-message](https://www.npmjs.com/package/react-native-toast-message)
- [react-native-bouncy-checkbox](https://github.com/WrathChaos/react-native-bouncy-checkbox)
